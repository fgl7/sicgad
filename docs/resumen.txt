Asunto: Demo institucional SICGAD para ABEN

El Sistema Integral de Carga, Gestión, Validación y Análisis de Datos Institucionales (SICGAD) se encuentra preparado como entorno demostrativo para la Agencia Boliviana de Energía Nuclear (ABEN).

La demo presenta una plataforma institucional para centralizar información, validar datos oficiales, reducir procesos manuales, generar indicadores ejecutivos y mantener trazabilidad sobre acciones relevantes. El objetivo es mostrar cómo la ABEN puede contar con información validada para seguimiento de servicios nucleares, proyectos estratégicos, mantenimiento de equipos críticos, seguridad radiológica, calidad y cumplimiento.

La demostración permite:

- administrar usuarios y accesos según rol;
- organizar la estructura institucional por sector, subsector, categoría y entidad;
- definir esquemas de datos institucionales;
- realizar cargas periódicas, históricas y manuales;
- validar y publicar información oficial;
- consultar indicadores ejecutivos según permisos;
- mantener trazabilidad y auditoría de acciones relevantes;
- revisar proyectos y convenios estratégicos.
