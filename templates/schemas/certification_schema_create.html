Crear esquema de certificación

Como Administrador puedes definir un esquema de datos mensuales para certificación, a partir de un dataset diario existente configurado por los Cargadores.

Selecciona primero el dataset base (diario) y luego marca los campos que quieres incluir en la certificación (por ejemplo: fecha, producto, producción). El esquema resultante se validará en los flujos habituales de validación hasta declararse producción certificada.
